pyCalc
A simple graphing calculator built on Python
As my first side project, I created a program that computes mathematical expressions and draw functions, much like a graphing calculator can.
It works by using a Read-Evaluate-Print Loop (REPL) that reads the user input and parse it into tokens, which represent objects to be evaluated. The tokens are sorted into a tree based on the order of operations. The tree is then evaluated by recursively evaluating the branches first. The final output is printed by the program and the whole process is restarted.

SPECIAL CASES
pyCalc supports some special functions. Three major ones are listed below. Try them out youself!
Implicit Multiplication
In mathematical syntax, we often do what is called multicplication by juxtaposition aka implicit multicplication, such as 5x or 5(4). The proximity of the two numbers imply mulitplication.

Try this:
5(4)
f(x) = 3x
Symbols
Some constants are just too long to write out. pyCalc supports substituting these numbers for symbols. Currently pi and euler's constant e are supported, but more can be added at any time.

Try this:
pi(4^2)
f(x) = e^x
Complex Numbers
While the browser implemention cannot support complex numbers, pyCalc actually can! They are reprenseted using the form a + bj where a is the real number and b is the imaginary number.

Try this:
(-5)^(0.5)
f(x) = (9-x^2)^0.5
